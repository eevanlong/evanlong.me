# CSS 效果实现

## CSS3 毛玻璃效果

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <link rel="stylesheet" href="index.css" />
    <title>CSS3毛玻璃</title>
  </head>
  <body>
    <div class="container">
      <div class="title">你好</div>
    </div>
  </body>
</html>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body,
.container {
  background: url(./image/vue3.png) 0 / cover fixed;
}

.container {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  width: 800px;
  height: 500px;
  border-radius: 5px;
  background: rgba(255, 255, 255, 0.3);
  backdrop-filter: blur(20px);
}
```

## 鼠标悬浮回显效果

```css
transition: all 0.5s;
&:hover {
  transform: translateY(-3px);
  transition: all 0.3s;
}
```
